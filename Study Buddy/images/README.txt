STUDY BUDDY - PIXEL ART ASSET REQUIREMENTS

This document outlines the required image assets for the Study Buddy Chrome extension.
All images should be created in a consistent 16-bit pixel art style.

REQUIRED IMAGES:

1. EXTENSION ICONS
   - icon-16.png (16×16px) - Small extension icon
   - icon-48.png (48×48px) - Medium extension icon
   - icon-128.png (128×128px) - Large extension icon
   Design: A pixel art hourglass or timer with the retro aesthetic

2. INTERFACE PATTERNS
   - dither-light.png (32×32px, tileable) - Light mode background pattern
     Colors: Use warm autumnal palette (#D35400, #A93226, #5D4037) with dithering effect
   
   - dither-dark.png (32×32px, tileable) - Dark mode background pattern
     Colors: Use nighttime sky theme (#1A2A3A, #8E44AD) with subtle star-pixel accents

3. PRESET ICONS (24×24px each)
   - pomodoro-icon.png - A pixel art tomato or timer split in 25/5 format
   - fifty-ten-icon.png - A 50/10 timer visualization
   - ninety-twenty-icon.png - A 90/20 timer visualization
   - custom-icon.png - A gear or wrench icon

4. UI ELEMENTS
   - settings-icon.png (16×16px) - A pixel art gear/cog

STYLE GUIDELINES:
- Use limited color palettes appropriate for 16-bit era graphics
- Each pixel should be distinct and visible (no anti-aliasing)
- Use dithering patterns for shading and texture
- Add pixel-perfect 1px highlights and shadows
- Maintain consistent pixel size and alignment

EXPORT SETTINGS:
- Save all files as PNG with transparency where needed
- Do not use compression that would blur pixel edges
- Ensure final images maintain crisp pixel boundaries
